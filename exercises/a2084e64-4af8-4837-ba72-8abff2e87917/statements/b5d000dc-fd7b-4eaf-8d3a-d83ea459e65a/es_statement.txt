<h2>Renombrar columnas de una tabla</h2><p>Si utilizamos la cl&aacute;usula <code>RENAME COLUMN</code> dentro de una sentencia <code>ALTER TABLE</code>, podremos modificar el nombre de una columna.</p><h3>Sintaxis:</h3><blockquote><p><code>ALTER TABLE TableName RENAME [ COLUMN ] <em>column_name</em> TO <em>new_column_name</em>;</code></p></blockquote><h3>Ejemplo:</h3><p>Hemos generado la tabla STUDENTS con la siguiente sentencia:</p><blockquote><pre><code>CREATE TABLE STUDENTS (<br />    LAST_NAME VARCHAR(50) NOT NULL,<br />    FIRST_NAME VARCHAR(25) NOT NULL,<br />    BIRTHDAY DATE,<br />    COURSE INTEGER<br />);</code></pre></blockquote><p>Queremos renombrar la columna <code>BIRTHDAY</code> para que se llame <code>BIRTH_DATE</code>, para lo que ejecutaremos la siguiente sentencia:</p><blockquote><pre><code>ALTER TABLE STUDENTS<br />    RENAME COLUMN BIRTHDAY TO BIRTH_DATE;</code></pre></blockquote><h3>Ejercicio:</h3><p>Imagina que hemos creado la siguiente tabla:</p><blockquote><pre><code>CREATE TABLE MODULES (<br />&nbsp;&nbsp;&nbsp; Name VARCHAR(100),<br />&nbsp;&nbsp;&nbsp; NumHours INTEGER,<br />&nbsp;&nbsp;&nbsp; Code NUMERIC(4),<br />&nbsp;&nbsp;&nbsp; Skill VARCHAR(100)<br />);</code></pre></blockquote><p>Modifica el nombre del atributo <code>NumHours</code> por el de <code>Number_Hours</code>.</p>
