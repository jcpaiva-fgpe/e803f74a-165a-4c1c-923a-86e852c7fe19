<h2>Renomear colunas de uma tabela</h2><p>Se usarmos a cláusula <code>RENAME COLUMN</code> dentro de uma instrução <code>ALTER TABLE</code>, podemos modificar o nome de uma coluna </p><h3>Sintaxe:</h3><blockquote><p><code>ALTER TABLE TableName RENAME [ COLUMN ] <em>column_name</em> TO <em>new_column_name</em>;</ code ></p></blockquote><h3>Exemplo:</h3><p>Geramos a tabela STUDENTS com a seguinte instrução:</p><blockquote><pre><code>CREATE TABLE STUDENTS ( <br /> LAST_NAME VARCHAR(50) NOT NULL,<br /> FIRST_NAME VARCHAR(25) NOT NULL,<br /> BIRTHDAY DATE,<br /> COURSE INTEGER<br />);</code></pre ></blockquote><p>Queremos renomear a coluna <code>BIRTHDAY</code> para que se chame <code>BIRTH_DATE</code>, para a qual executaremos a seguinte instrução:</p>< blockquote><pre>< code>ALTER TABLE STUDENTS<br /> RENAME COLUMN BIRTHDAY TO BIRTH_DATE;</code></pre></blockquote><h3>Exercício:</h3><p>Imagine que criamos a tabela a seguir:</p ><blockquote><pre><code> CREATE TABLE MODULES (<br /> Nome VARCHAR(100),<br /> NumHours INTEGER,<br /> Código NUMERIC(4),<br /> Skill VARCHAR(100)<br />);</code> </pre></blockquote><p>Altere o nome do atributo <code>NumHours</code> para <code>Number_Hours</code>.</p>
