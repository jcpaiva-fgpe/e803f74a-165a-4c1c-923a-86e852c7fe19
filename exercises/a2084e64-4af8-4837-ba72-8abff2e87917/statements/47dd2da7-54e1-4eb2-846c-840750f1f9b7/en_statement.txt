<h2>Rename columns of a table</h2><p>If we use the <code>RENAME COLUMN</code> clause within an <code>ALTER TABLE</code> statement, we can modify the name of a column. </p><h3>Syntax:</h3><blockquote><p><code>ALTER TABLE TableName RENAME [ COLUMN ] <em>column_name</em> TO <em>new_column_name</em>;</code ></p></blockquote><h3>Example:</h3><p>We have generated the STUDENTS table with the following statement:</p><blockquote><pre><code>CREATE TABLE STUDENTS (<br /> LAST_NAME VARCHAR(50) NOT NULL,<br /> FIRST_NAME VARCHAR(25) NOT NULL,<br /> BIRTHDAY DATE,<br /> COURSE INTEGER<br />);</code></pre>< /blockquote><p>We want to rename the column <code>BIRTHDAY</code> so that it is called <code>BIRTH_DATE</code>, for which we will execute the following statement:</p><blockquote><pre>< code>ALTER TABLE STUDENTS<br /> RENAME COLUMN BIRTHDAY TO BIRTH_DATE;</code></pre></blockquote><h3>Exercise:</h3><p>Imagine that we have created the following table:</p ><blockquote><pre><code> CREATE TABLE MODULES (<br /> Name VARCHAR(100),<br /> NumHours INTEGER,<br /> Code NUMERIC(4),<br /> Skill VARCHAR(100)<br />);</code> </pre></blockquote><p>Change the name of the <code>NumHours</code> attribute to <code>Number_Hours</code>.</p>
