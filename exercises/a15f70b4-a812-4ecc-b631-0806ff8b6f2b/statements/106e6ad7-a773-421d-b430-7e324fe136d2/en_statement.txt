<h2>Modify columns of a table</h2><p>If we want to modify columns of a table: we can modify both the data type and its properties. All changes are possible if the table contains no data. In general, if the table is not empty we can increase the length of a column, increase or decrease the number of decimal places in a NUMERIC type, reduce the width as long as the data does not occupy all the space reserved for it.</p> <h3>Syntax:</h3><blockquote><p><code>ALTER TABLE TableName ALTER [ COLUMN ] <em>column_name</em> [ SET DATA ] TYPE <em>data_type</em>;</code ></p></blockquote><h3>Example:</h3><p>In the previous exercise we generated the <code>USERS</code> table with the following structure:</p><blockquote>< pre> table_name | column_name | data_type | max_length <br />------------+-------------+------------------ -+------------<br /> users | birth_date | date | <br /> users | course_code | integer | 32<br /> users | first_name | character varying | 25 users | last_name | character varying | 50</pre></blockquote><p>This structure is equivalent to executing the following statement:</p><blockquote><pre><code>CREATE TABLE USERS (<br /> FIRST_NAME VARCHAR(25) ,<br /> LAST_NAME VARCHAR(50),<br /> Birth_Date DATE,<br /> Course_Code INTEGER<br />);</code></pre></blockquote><p>We noticed that the size reserved for both the <em>Name</em> and the <em>Surnames</em> is too large, so, to save space, we have decided to reduce the size of these attributes to 15 and 30 respectively . To do this, we should execute the following statement:</p><blockquote><pre><code>ALTER TABLE USERS<br /> ALTER FIRST_NAME TYPE VARCHAR(15),<br /> ALTER LAST_NAME TYPE VARCHAR(30);< /code></pre></blockquote><h3>Exercise:</h3><p>Imagine that we have created the following table:</p><blockquote><pre><code>CREATE TABLE MODULES (<br /> Name VARCHAR(100),<br /> NumHours INTEGER,<br /> Code NUMERIC(4),<br /> Skill VARCHAR(100)<br />);</code></pre></ blockquote><p>Increases the number of digits that the <code>Code</code> attribute can contain up to 5 and reduces the <code>Skill</code> attribute to a maximum of 60 characters.</p><p > </p>
