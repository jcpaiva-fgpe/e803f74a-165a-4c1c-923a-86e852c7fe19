<h2>Modificar columnas de una tabla</h2><p>Si queremos modificar columnas de una tabla: podemos modificar tanto el tipo de datos como sus propiedades. Todos los cambios son posibles si la tabla no contiene datos. En general, si la tabla no est&aacute; vac&iacute;a podremos aumentar la longitud de una columna, aumentar o disminuir el n&uacute;mero de posiciones decimales en un tipo NUMERIC, reducir la anchura siempre que los datos no ocupen todo el espacio reservado para ellos.</p><h3>Sintaxis:</h3><blockquote><p><code>ALTER TABLE TableName ALTER [ COLUMN ] <em>column_name</em> [ SET DATA ] TYPE <em>data_type</em>;</code></p></blockquote><h3>Ejemplo:</h3><p>En el ejercicio anterior hemos generado la tabla <code>USERS</code> con la siguiente estructura:</p><blockquote><pre> table_name | column_name |     data_type     | max_length <br />------------+-------------+-------------------+------------<br /> users      | birth_date  | date              |           <br /> users      | course_code | integer           |         32<br /> users      | first_name  | character varying |         25<br /> users      | last_name   | character varying |         50</pre></blockquote><p>Esta estructura es equivalente a la ejecuci&oacute;n de la siguiente sentencia:</p><blockquote><pre><code>CREATE TABLE USERS (<br />    FIRST_NAME VARCHAR(25),<br />    LAST_NAME VARCHAR(50),<br />    Birth_Date DATE,<br />    Course_Code INTEGER<br />);</code></pre></blockquote><p>Nos hemos dado cuenta de que el tama&ntilde;o reservado tanto al <em>Nombre</em> como a los <em>Apellidos</em> es demasiado grande, por lo que, para ahorrar espacio, hemos decidido reducir el tama&ntilde;o de dichos atributos a 15 y 30 respectivamente. Para ello, deber&iacute;amos ejecutar la siguiente sentencia:</p><blockquote><pre><code>ALTER TABLE USERS<br />    ALTER FIRST_NAME TYPE VARCHAR(15),<br />    ALTER LAST_NAME TYPE VARCHAR(30);</code></pre></blockquote><h3>Ejercicio:</h3><p>Imagina que hemos creado la siguiente tabla:</p><blockquote><pre><code>CREATE TABLE MODULES (<br />&nbsp;&nbsp;&nbsp; Name VARCHAR(100),<br />&nbsp;&nbsp;&nbsp; NumHours INTEGER,<br />&nbsp;&nbsp;&nbsp; Code NUMERIC(4),<br />&nbsp;&nbsp;&nbsp; Skill VARCHAR(100)<br />);</code></pre></blockquote><p>Aumenta el n&uacute;mero de d&iacute;gitos que puede contener el atributo <code>Code</code> hasta 5 y reduce el atributo <code>Skill</code> a un m&aacute;ximo de 60 caracteres.</p><p>&nbsp;</p>
