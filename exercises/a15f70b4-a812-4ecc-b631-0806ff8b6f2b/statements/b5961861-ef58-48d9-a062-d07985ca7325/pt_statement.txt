<h2>Modificar colunas de uma tabela</h2><p>Se quisermos modificar colunas de uma tabela: podemos modificar tanto o tipo de dados quanto suas propriedades. Todas as alterações são possíveis se a tabela não contiver dados. Em geral, se a tabela não estiver vazia podemos aumentar o comprimento de uma coluna, aumentar ou diminuir o número de casas decimais em um tipo NUMERIC, reduzir a largura desde que os dados não ocupem todo o espaço reservado para ele.< /p> <h3>Sintaxe:</h3><blockquote><p><code>ALTER TABLE TableName ALTER [ COLUMN ] <em>column_name</em> [ SET DATA ] TYPE <em>data_type</em>; </code ></p></blockquote><h3>Exemplo:</h3><p>No exercício anterior geramos a tabela <code>USERS</code> com a seguinte estrutura:</p>< blockquote>< pre> table_name | nome_coluna | data_type | max_length <br />------------+-------------+------------------- -+------------<br /> usuários | data_nascimento | data | <br /> usuários | course_code | inteiro | 32<br /> usuários | first_name | personagem variando | 25 usuários | sobrenome | personagem variando | 50</pre></blockquote><p>Esta estrutura é equivalente a executar a seguinte instrução:</p><blockquote><pre><code>CREATE TABLE USERS (<br /> FIRST_NAME VARCHAR(25) ,< br /> LAST_NAME VARCHAR(50),<br /> Birth_Date DATE,<br /> Course_Code INTEGER<br />);</code></pre></blockquote><p>Percebemos que o tamanho reservado para tanto o <em>Nome</em> quanto o <em>Sobrenomes</em> são muito grandes, então, para economizar espaço, decidimos reduzir o tamanho desses atributos para 15 e 30 respectivamente . Para fazer isso, devemos executar a seguinte instrução:</p><blockquote><pre><code>ALTER TABLE USERS<br /> ALTER FIRST_NAME TYPE VARCHAR(15),<br /> ALTER LAST_NAME TYPE VARCHAR(30) ;</code></pre></blockquote><h3>Exercício:</h3><p>Imagine que criamos a seguinte tabela:</p><blockquote><pre><code>CREATE TABLE MODULES (<br /> Nome VARCHAR(100),<br /> NumHours INTEGER,<br /> Código NUMERIC(4),<br /> Habilidade VARCHAR(100)<br />);</code></pre ></ blockquote><p>Aumenta o número de dígitos que o atributo <code>Code</code> pode conter até 5 e reduz o atributo <code>Habilidade</code> para um máximo de 60 caracteres.</code> p><p > </p>
