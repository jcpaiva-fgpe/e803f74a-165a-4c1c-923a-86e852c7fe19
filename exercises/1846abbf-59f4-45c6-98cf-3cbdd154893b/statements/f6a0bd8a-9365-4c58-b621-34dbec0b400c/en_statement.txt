<h2><code>DROP TABLE</code></h2><p>When a table is no longer useful and we don't need it, it's convenient to delete it; in this way it will not take up space and we can use its name on another occasion.</p><p>To delete a table we will use the command <a href="https://www.postgresql.org/docs/14/sql- droptable.html" target="_blank"><code>DROP TABLE</code></a>.</p><h3>Syntax:</h3><blockquote><pre><code>DROP TABLE TableName [ CASCADE];</code></pre></blockquote><p>This statement will delete the database table including its data (rows). All the information that existed in that table in the Data Dictionary will also be deleted.</p><h3>Example:</h3><p>If we have created a <code>USERS</code> table with the command: <blockquote><pre><code>CREATE TABLE USERS (<br /> Login VARCHAR (25)<br /> CONSTRAINT Users_log_PK PRIMARY KEY<br />);</code></pre></blockquote><p >We could remove it with the following command:</p><blockquote><pre><code>DROP TABLE USERS;</code></pre></blockquote><p>Be careful when using this command, deletion of a table is irreversible and there is no confirmation prompt before it is executed.</p><h3>Exercise:</h3><p>Now imagine we have created the <code>Users</code> and <code> tables Modules</code> with the following code:</p><blockquote><pre><code>CREATE TABLE USERS (<br /> Login VARCHAR (25)<br /> CONSTRAINT Users_log_PK PRIMARY KEY<br />); <br /><br />CREATE TABLE MODULES (<br /> name VARCHAR(100) PRIMARY KEY,<br /> hours INTEGER<br />);</code></pre></blockquote><p >How would you delete the table <code>MODULES</code>?</p>
