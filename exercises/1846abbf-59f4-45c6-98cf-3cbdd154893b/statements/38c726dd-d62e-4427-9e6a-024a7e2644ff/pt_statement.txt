<h2><code>DROP TABLE</code></h2><p>Quando uma tabela não é mais útil e não precisamos dela, é conveniente excluí-la; desta forma não ocupará espaço e poderemos utilizar seu nome em outra ocasião.</p><p>Para deletar uma tabela utilizaremos o comando <a href="https://www.postgresql.org/ docs/14/sql-droptable.html" target="_blank"><code>DROP TABLE</code></a>.</p><h3>Sintaxe:</h3><blockquote><pre>< code>DROP TABLE TableName [ CASCADE];</code></pre></blockquote><p>Esta instrução irá deletar a tabela do banco de dados incluindo seus dados (linhas). Todas as informações que existiam naquela tabela no Dicionário de Dados também serão deletadas.</p><h3>Exemplo:</h3><p>Se criamos uma tabela <code>USUÁRIOS</code> com o comando : <blockquote><pre><code>CREATE TABLE USERS (<br /> Login VARCHAR (25)<br /> CONSTRAINT Users_log_PK PRIMARY KEY<br />);</code></pre></blockquote>< p >Podemos removê-lo com o seguinte comando:</p><blockquote><pre><code>DROP TABLE USERS;</code></pre></blockquote><p>Tenha cuidado ao usar este comando, a exclusão de uma tabela é irreversível e não há prompt de confirmação antes de ser executada.</p><h3>Exercício:</h3><p>Agora imagine que criamos <code>Users</code> e <code > Tables Modules</code> com o seguinte código:</p><blockquote><pre><code>CREATE TABLE USERS (<br /> Login VARCHAR (25)<br /> CONSTRAINT Users_log_PK PRIMARY KEY<br /> ); <br /><br />CRIA MÓDULOS DE TABELA (<br /> nome VARCHAR(100) CHAVE PRIMÁRIA,<br /> horas INTEGER<br />);</code></pre></blockquote> <p>Como você excluiria a tabela <code>MODULES</code>?</p>
