<h2><code>DROP TABLE</code></h2><p>Cuando una tabla ya no es &uacute;til y no la necesitamos es conveniente borrarla; de este modo no ocupar&aacute; espacio y podremos utilizar su nombre en otra ocasi&oacute;n.</p><p>Para eliminar una tabla utilizaremos el comando <a href="https://www.postgresql.org/docs/14/sql-droptable.html" target="_blank"><code>DROP TABLE</code></a>.</p><h3>Sintaxis:</h3><blockquote><pre><code>DROP TABLE TableName [CASCADE];</code></pre></blockquote><p>Esta instrucci&oacute;n borrar&aacute; la tabla de la base de datos incluido sus datos (filas). Tambi&eacute;n se borrar&aacute; toda la informaci&oacute;n que existiera de esa tabla en el Diccionario de Datos.</p><h3>Ejemplo:</h3><p>Si hemos creado una tabla <code>USERS</code> con el comando:<blockquote><pre><code>CREATE TABLE USERS (<br />     Login VARCHAR (25)<br />          CONSTRAINT Users_log_PK PRIMARY KEY<br />);</code></pre></blockquote><p>Podr√≠amos eliminarla con el siguiente comando:</p><blockquote><pre><code>DROP TABLE USERS;</code></pre></blockquote><p>Ten cuidado al utilizar este comando, el borrado de una tabla es irreversible y no hay una petici&oacute;n de confirmaci&oacute;n antes de ejecutarse.</p><h3>Ejercicio:</h3><p>Ahora imagina que hemos creado las tablas <code>Users</code> y <code>Modules</code> con el siguiente c&oacute;digo:</p><blockquote><pre><code>CREATE TABLE USERS (<br />     Login VARCHAR (25)<br />          CONSTRAINT Users_log_PK PRIMARY KEY<br />);<br /><br />CREATE TABLE MODULES (<br />    name VARCHAR(100) PRIMARY KEY,<br />    hours INTEGER<br />);</code></pre></blockquote><p>&iquest;C&oacute;mo borrar&iacute;as la tabla <code>MODULES</code>?</p>
