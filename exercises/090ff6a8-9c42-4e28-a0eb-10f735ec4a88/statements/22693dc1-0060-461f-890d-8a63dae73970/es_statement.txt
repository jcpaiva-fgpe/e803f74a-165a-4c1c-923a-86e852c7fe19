<h2>Modificar una tabla. Restricciones de tabla.</h2><p>Tambi&eacute;n podemos a&ntilde;adir restricciones de tabla, obligatorias cuando la restricci&oacute;n afecta a m&aacute;s de una columna. Por ejemplo, cuando la clave primaria o la clave ajena est&aacute; formada por m&aacute;s de un atributo.</p><h3>Sintaxis:</h3><blockquote><p><code>ALTER TABLE TableName ADD <em>table_constraint</em>;</code></p></blockquote><p>Donde <em>table_constraint</em> podr&iacute;a ser de tipo <code>UNIQUE</code>, <code>PRIMARY KEY</code>, <code>FOREIGN KEY</code> o <code>CHECK</code>.</p><h3>Ejemplo:</h3><p>Imaginemos que hemos ejecutado la siguiente sentencia de creaci&oacute;n de tabla:</p><blockquote><pre><code>CREATE TABLE STUDENTS (<br />    LAST_NAME VARCHAR(50) NOT NULL,<br />    FIRST_NAME VARCHAR(25) NOT NULL,<br />    BIRTH_DATE DATE,<br />    COURSE INTEGER<br />);</code></pre></blockquote><p>Pero, es necesario incorporar, como clave primaria, el conjunto de los atributos <code>LAST_NAME</code>, <code>FIRST_NAME</code> y <code>BIRTH_DATE</code>:</p><blockquote><code>ALTER TABLE STUDENTS<br />&nbsp;&nbsp;&nbsp; ADD PRIMARY KEY (LAST_NAME, FIRST_NAME, BIRTH_DATE);</code></blockquote><h3>Ejercicio:</h3><p>Imagina que hemos creado las siguientes tablas:</p><blockquote><p><code>CREATE TABLE MODULES (<br />&nbsp;&nbsp;&nbsp; CODE INTEGER,<br />&nbsp;&nbsp;&nbsp; NAME VARCHAR(100),<br />&nbsp;&nbsp;&nbsp; DEGREE CHAR(3)<br />);</code></p><p><code>CREATE TABLE BOOKS (<br />&nbsp;&nbsp;&nbsp; ISBN NUMERIC,<br />&nbsp;&nbsp;&nbsp; TITLE VARCHAR(100),<br />&nbsp;&nbsp;&nbsp; MODULE INTEGER,</code><br /><code>&nbsp;&nbsp;&nbsp; DEGREE CHAR(3)<br />);</code></p></blockquote><p>Incorpora las siguientes restricciones:</p><ul><li>La clave primaria de la tabla <code>MODULES</code> debe ser el conjunto de <code>CODE</code> y <code>DEGREE</code>.</li><li>La clave primaria de la tabla <code>BOOKS</code> debe ser el atributo <code>ISBN</code>.</li><li>El conjunto (<code>MODULE</code>, <code>DEGREE</code>) de la tabla <code>BOOKS</code> referencia a la tabla <code>MODULES</code>.</li></ul><p>&nbsp;</p>
