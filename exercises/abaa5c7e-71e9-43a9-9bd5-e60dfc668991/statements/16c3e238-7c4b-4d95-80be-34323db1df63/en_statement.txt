<h2>Clause <code>ON DELETE</code>.</h2><p>The <code>ON DELETE</code> clause allows you to determine the way in which the Database maintains referential integrity when deleting a primary or unique key. If we do not use the <code>ON DELETE</code> clause, the Database manager does not allow a key referenced in the <em>parent</em> table to be deleted if it has any row in the <em> table. daughter</em> that references it.</p><h3>Exercise:</h3><p>Modify the solution of the previous exercise, so that when a record is deleted from the table <code>MODULES</code > all records in the BOOKS table that are related to the deleted record are automatically deleted.</p>
