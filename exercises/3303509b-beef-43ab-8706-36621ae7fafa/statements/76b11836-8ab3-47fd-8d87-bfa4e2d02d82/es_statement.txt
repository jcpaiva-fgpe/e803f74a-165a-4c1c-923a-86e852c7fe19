<h2>A&ntilde;adir columnas a una tabla</h2><p>Utilizaremos la cl&aacute;usula <code>ADD</code> del comando <code>ALTER TABLE</code>, junto con la definici&oacute;n de las nuevas columnas. Las columnas se a&ntilde;adir&aacute;n al final de la tabla.</p><h3>Sintaxis:</h3><blockquote><pre><code>ALTER TABLE TableName <strong>ADD</strong> [ COLUMN ] [ IF NOT EXISTS ] <em><code>column_name</code></em> <em><code>data_type</code></em> [ <em><code>column_constraint</code></em> [ ... ] ];</code></pre></blockquote><h3>Ejemplo:</h3><p>En un ejercicio anterior hemos creado la tabla <code>USERS</code> con la siguiente sentencia:</p><blockquote><pre><code>CREATE TABLE USERS (<br />    FIRST_NAME VARCHAR(25),<br />    LAST_NAME VARCHAR(30)<br />);</code></pre></blockquote><p>Posteriormente, necesitamos a&ntilde;adir las dos siguientes columnas:</p><ul><li><code>Birth_Date</code>: de tipo fecha.</li><li><code>Course_Code</code>: n&uacute;mero sin decimales.</li></ul><p>Deber&iacute;amos ejecutar la siguiente sentencia:</p><blockquote><pre><code>ALTER TABLE USERS<br />    ADD Birth_Date DATE,<br />    ADD Course_Code INTEGER;</code></pre></blockquote><h3>Ejercicio:</h3><p>Imagina que hemos creado la siguiente tabla:</p><blockquote><pre><code>CREATE TABLE MODULES (<br />    Name VARCHAR(100),<br />    NumHours INTEGER<br />);</code></pre></blockquote><p>A&ntilde;ade los siguientes atributos a la tabla:</p><ul><li>Code: Num&eacute;rico de 4 d√≠gitos.</li><li>Skill: Texto de hasta 100 caracteres.</li></ul>
