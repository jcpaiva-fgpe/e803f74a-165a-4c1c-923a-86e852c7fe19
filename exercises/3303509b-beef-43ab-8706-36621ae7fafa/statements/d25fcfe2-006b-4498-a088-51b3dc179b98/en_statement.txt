<h2>Adding columns to a table</h2><p>We will use the <code>ADD</code> clause of the <code>ALTER TABLE</code> command, along with the definition of the new columns. The columns will be added to the end of the table.</p><h3>Syntax:</h3><blockquote><pre><code>ALTER TABLE TableName <strong>ADD</strong> [ COLUMN ] [ IF NOT EXISTS ] <em><code>column_name</code></em> <em><code>data_type</code></em> [ <em><code>column_constraint</code></em> [ .. ] ];</code></pre></blockquote><h3>Example:</h3><p>In a previous exercise we created the <code>USERS</code> table with the following statement:< /p><blockquote><pre><code>CREATE TABLE USERS (<br /> FIRST_NAME VARCHAR(25),<br /> LAST_NAME VARCHAR(30)<br />);</code></pre>< /blockquote><p>Next, we need to add the following two columns:</p><ul><li><code>Birth_Date</code>: of type date.</li><li><code>Course_Code< /code>: number without decimals.</li></ul><p>We should execute the following statement:</p><blockquote><pre><code>ALTER TABLE USERS<br /> ADD Birth_Date DATE,< br /> ADD Course_Code INTEGER;</code></pre></blockquote><h3>Exercise:</h3><p>Imagine that we have created the following table:</p><b lockquote><pre><code>CREATE TABLE MODULES (<br /> Name VARCHAR(100),<br /> NumHours INTEGER<br />);</code></pre></blockquote><p>Add the following attributes to the table:</p><ul><li>Code: Numeric with 4 digits.</li><li>Skill: Text with up to 100 characters.</li></ul>
