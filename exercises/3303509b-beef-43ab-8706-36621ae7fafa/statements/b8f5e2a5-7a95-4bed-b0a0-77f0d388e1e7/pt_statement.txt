<h2>Adicionando colunas a uma tabela</h2><p>Usaremos a cláusula <code>ADD</code> do comando <code>ALTER TABLE</code>, juntamente com a definição das novas colunas. As colunas serão adicionadas ao final da tabela.</p><h3>Sintaxe:</h3><blockquote><pre><code>ALTER TABLE TableName <strong>ADD</strong> [ COLUMN ] [ IF NÃO EXISTE ] <em><code>column_name</code></em> <em><code>data_type</code></em> [ <em><code>column_constraint</code></em> [ .. ] ];</code></pre></blockquote><h3>Exemplo:</h3><p>Em um exercício anterior criamos a tabela <code>USERS</code> com a seguinte instrução: </p><blockquote><pre><code>CREATE TABLE USERS (<br /> FIRST_NAME VARCHAR(25),<br /> LAST_NAME VARCHAR(30)<br />);</code></pre> </blockquote><p>Em seguida, precisamos adicionar as duas colunas a seguir:</p><ul><li><code>Birth_Date</code>: do tipo date.</li><li><code >Course_Code</code>: número sem decimais.</li></ul><p>Devemos executar a seguinte instrução:</p><blockquote><pre><code>ALTER TABLE USERS<br /> ADD Birth_Date DATE,< br /> ADD Course_Code INTEGER;</code></pre></blockquote><h3>Exercise:</h3><p>Imagine que criamos a seguinte tabela:</p><b lockquote><pre><code>CREATE TABLE MODULES (<br /> Name VARCHAR(100),<br /> NumHours INTEGER<br />);</code></pre></blockquote><p>Add os seguintes atributos à tabela:</p><ul><li>Código: Numérico com 4 dígitos.</li><li>Habilidade: Texto com até 100 caracteres.</li></ul>
