<h2>Delete indexes.</h2><p>Once created, we can delete indexes, through the name chosen at the time of creation.</p><h3>Syntax:</h3><blockquote>< pre><code>DROP INDEX IndexName;</code></pre></blockquote><h3>Example:</h3><p>We have created the following table:</p><blockquote><pre>< code>CREATE TABLE MY_EMPLOYEES (<br /> COD_EMPLOYEE VARCHAR(5) PRIMARY KEY,<br /> FIRST_NAME VARCHAR(15),<br /> LAST_NAME VARCHAR(30),<br /> SALARY NUMERIC(6,2)< br />);</code></pre></blockquote><p>and the index</p><blockquote><pre><code>CREATE INDEX employeesLastNameIndex ON MY_EMPLOYEES (LAST_NAME);</code>< /pre></blockquote><p>To delete the index we write the following statement:</p><blockquote><pre><code>DROP INDEX employeesLastNameIndex;</code></pre></blockquote><h3 >Exercise:</h3><p>Imagine that we have created the following table:</p><blockquote><pre><code>CREATE TABLE MODULES (<br /> Name VARCHAR(100),<br /> NumHours INTEGER,<br /> Code NUMERIC(4),<br /> Skill VARCHAR(100 )<br />);</code></pre></blockquote><p>and an index with the following statement:</p><blockquote><pre><code>CREATE INDEX modulesNameIndex ON modules (name );</code></pre></blockquote><p>Delete the newly created index.</p>
