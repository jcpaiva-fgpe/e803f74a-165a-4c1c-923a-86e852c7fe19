<h2>Excluir índices.</h2><p>Uma vez criados, podemos excluir índices, através do nome escolhido no momento da criação.</p><h3>Sintaxe:</h3><blockquote>< pre> <code>DROP INDEX IndexName;</code></pre></blockquote><h3>Exemplo:</h3><p>Criamos a seguinte tabela:</p><blockquote><pre>< code >CREATE TABLE MY_EMPLOYEES (<br /> COD_EMPLOYEE VARCHAR(5) PRIMARY KEY,<br /> FIRST_NAME VARCHAR(15),<br /> LAST_NAME VARCHAR(30),<br /> SALARY NUMERIC(6,2)< br />);</code></pre></blockquote><p>e o índice</p><blockquote><pre><code>CREATE INDEX employeesLastNameIndex ON MY_EMPLOYEES (LAST_NAME);</code></ pre></blockquote><p>Para excluir o índice, escrevemos a seguinte frase:</p><blockquote><pre><code>DROP INDEX employeesLastNameIndex;</code></pre></blockquote><h3 >Exercício:</h3><p>Imagine que criamos a seguinte tabela:</p><blockquote><pre><code>CREATE TABLE MODULES (<br /> Name VARCHAR(100),<br /> NumHours INTEGER,<br /> Código NUMERIC(4),<br /> Habilidade VARCHAR(100 )<br />);</code></pre></blockquote><p>e um índice com a seguinte declaração:</p><blockquote><pre><code>CREATE INDEX modulesNameIndex ON modules (name );</code></pre></blockquote><p>Exclua o índice recém-criado.</p>
