<h2>Ta bort index.</h2><p>När de har skapats kan vi ta bort index, genom det namn som valdes vid tidpunkten för skapande.</p><h3>Syntax:</h3><blockquote>< pre> <code>DROP INDEX IndexName;</code></pre></blockquote><h3>Exempel:</h3><p>Vi har skapat följande tabell:</p><blockquote><pre>< code >SKAPA TABELL MY_EMPLOYEES (<br /> COD_EMPLOYEE VARCHAR(5) PRIMARY KEY,<br /> FIRST_NAME VARCHAR(15),<br /> LAST_NAME VARCHAR(30),<br /> LÖNNUMERISK(6,2)< br />);</code></pre></blockquote><p>och indexet</p><blockquote><pre><code>SKAPA INDEX anställdaLastNameIndex PÅ MY_EMPLOYEES (LAST_NAME);</code>< / pre></blockquote><p>För att ta bort indexet skriver vi följande mening:</p><blockquote><pre><code>DROP INDEX anställdaLastNameIndex;</code></pre></blockquote><h3 >Övning:</h3><p>Föreställ dig att vi har skapat följande tabell:</p><blockquote><pre><code>SKAPA TABELLMODULER (<br /> Namn VARCHAR(100),<br /> Antal timmar INTEGER,<br /> Kod NUMERIC(4),<br /> Färdighet VARCHAR(100 )<br />);</code></pre></blockquote><p>och ett index med följande sats:</p><blockquote><pre><code>CREATE INDEX modulesNameIndex ON modules (namn );</code></pre></blockquote><p>Ta bort det nyskapade indexet.</p>
