<h2>Borrar &iacute;ndices.</h2><p>Una vez creados, podemos borrar &iacute;ndices, a trav&eacute;s del&nbsp; nombre elegido en el momento de la creaci&oacute;n.</p><h3>Sintaxis:</h3><blockquote><pre><code>DROP INDEX IndexName;</code></pre></blockquote><h3>Ejemplo:</h3><p>Tenemos creada la siguiente tabla:</p><blockquote><pre><code>CREATE TABLE MY_EMPLOYEES (<br />    COD_EMPLOYEE VARCHAR(5) PRIMARY KEY,<br />    FIRST_NAME VARCHAR(15),<br />    LAST_NAME VARCHAR(30),<br />    SALARY NUMERIC(6,2)<br />);</code></pre></blockquote><p>y el &iacute;ndice</p><blockquote><pre><code>CREATE INDEX employeesLastNameIndex ON MY_EMPLOYEES (LAST_NAME);</code></pre></blockquote><p>Para borrar el &iacute;ndice escribimos la siguiente sentencia:</p><blockquote><pre><code>DROP INDEX employeesLastNameIndex;</code></pre></blockquote><h3>Ejercicio:</h3><p>Imagina que hemos creado la siguiente tabla:</p><blockquote><pre><code>CREATE TABLE MODULES (<br />&nbsp;&nbsp;&nbsp; Name VARCHAR(100),<br />&nbsp;&nbsp;&nbsp; NumHours INTEGER,<br />&nbsp;&nbsp;&nbsp; Code NUMERIC(4),<br />&nbsp;&nbsp;&nbsp; Skill VARCHAR(100)<br />);</code></pre></blockquote><p>y un &iacute;ndice con la siguiente sentencia:</p><blockquote><pre><code>CREATE INDEX modulesNameIndex ON modules (name);</code></pre></blockquote><p>Borra el &iacute;ndice reci&eacute;n creado.</p>
