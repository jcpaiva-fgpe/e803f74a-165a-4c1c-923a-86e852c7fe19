<p>Os objetos básicos com os quais o <code>SQL</code> trabalha para armazenar os dados são as <strong>tabelas</strong>,
 que como já sabemos são um conjunto de <em>linhas</em> e <em>colunas</em> cuja interseção é chamada <em>célula</em>. É aqui que serão armazenados os elementos de informação, os dados que queremos coletar.</p><p>Antes de criar a tabela, convém planejar alguns detalhes:</p><ul><li>< strong>Que nome</strong> daremos <strong>à tabela</strong>.</li><li><strong>Que nome</strong> daremos a cada um dos <strong >colunas</strong >.</li><li><strong>Que tipo e tamanho de dados</strong> vamos armazenar em cada coluna.</li><li><strong>Quais restrições</strong> strong> que temos sobre os dados. </li><li>Quaisquer outras <strong>informações adicionais</strong> que precisamos.</li></ul><p>E devemos levar em consideração outras < strong>regras</strong> que devem ser atendidas <strong>para nomes de tabelas</strong>:</p><ul><li>Não podemos ter nomes de tabelas duplicados no mesmo esquema (usuário).</li> <li>Eles devem começar com um caractere alfabético.</li><li>O comprimento máximo é de 30 caracteres.</li><li>Somente letras, dígitos ou sublinhado em inglês são permitidos.</li><li> Você não pode para corresponder a palavras reservadas <code>SQL</code> (por exemplo, não podemos chamar uma tabela <code>WHERE</code>).</li><li>Não diferencia maiúsculas de minúsculas. </li><li >Caso o nome tenha espaços em branco ou caracteres nacionais (permitidos apenas em alguns bancos de dados), então geralmente é colocado entre aspas duplas.</li></ul><h3> Sintaxe</h3><p>A sintaxe básica do comando usado para criar uma tabela é o seguinte:</p><blockquote><pre><code>CREATE TABLE [schema.]tablename (column1 datatype, column2 datatype,. ..columnN datatype );</code> </pre></blockquote><p>onde:</p><ul><li><code>column1, column2, ..., columnN</code> são os nomes das colunas que a tabela conterá .</li><li><code>datatype</code> indica o tipo de dados de cada coluna.</li></ul><h3>Exemplo:</h3 ><p>Para criar uma tabela chamada < strong>ESTUDANTES</strong> com dois campos do tipo <code>VARCHAR</code>:</p><blockquote><pre><code>CREATE TABLE STUDENTS (<br /> firstName V ARCHAR(25),<br />    lastName VARCHAR(50)<br />);</code></pre></blockquote><h3>Ejercicio:</h3><p>Crea una tabla <code> MODULES</code>, compuesta de los siguientes atributos:</p><ul><li><code>name</code>: de tipo <code>VARCHAR</code> y tamaño 100.</li>< li><code>hours</code>: de tipo <code>INTEGER</code>.</li></ul><p><small><strong>Nota:</strong> Más adelante veremos como borrar Tabelas. No entanto, como você certamente terá que excluir tabelas para poder refazê-las corretamente ou com maior precisão, você deve usar o comando <code>DROP TABLE table_name;</code>. Por exemplo: <ul><li><code>DROP TABLE USERS;</code>,</li><li><code>DROP TABLE MODULES;</code> ou </li><li><code> DROP TABLE ALUNOS;</code></small></li></ul></p>
