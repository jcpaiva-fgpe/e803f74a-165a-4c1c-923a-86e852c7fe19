<h2>Removendo colunas de uma tabela</h2><p>Nós usaremos a cláusula <code>DROP</code> do comando <code>ALTER TABLE</code>, junto com a coluna que queremos remover. As colunas indicadas serão excluídas sem poder desfazer esta ação. Além da definição da coluna, todos os dados nela contidos serão excluídos. Uma coluna não pode ser excluída se for a única que forma a tabela, para isso teremos que excluir a tabela diretamente.</p><h3>Sintaxe:</h3><blockquote><pre><code>ALTER TABLE TableName DROP [ COLUMN ] [ IF EXISTS ] <em>column_name</em> [ RESTRICT | CASCADE ];</code></pre></blockquote><h3>Exemplo:</h3><p>No exercício anterior geramos a tabela <code>USERS</code> com a seguinte estrutura:</ p><blockquote><pre> table_name | nome_coluna | data_type <br />------------+-------------+---------------------- -<br /> usuários | data_nascimento | data<br /> usuários | course_code | inteiro<br /> usuários | first_name | caractere variando<br /> usuários | sobrenome | caractere variando<br />(4 linhas)</pre></blockquote><p>Esta estrutura é equivalente a executar a seguinte instrução:</p><blockquote><pre><code>CREATE TABLE USERS ( <br /> FIRST_NAME VARCHAR(25),<br /> LAST_NAME VARCHAR(30),<br /> Birth_Date DATE,<br /> Course_Code INTEGER<br />);</code></pre></blockquote >< p>Para remover os dois atributos adicionados na última seção, devemos executar a seguinte instrução:</p><blockquote><pre><code>ALTER TABLE USERS<br /> DROP Birth_Date,<br /> DROP Course_Code ; </code></pre></blockquote><h3>Exercício:</h3><p>Imagine que criamos a seguinte tabela:</p><blockquote><pre><code>CREATE TABLE MODULES ( <br /> Nome VARCHAR(100),<br /> NumHours NUMERIC(4),<br /> Código INTEGER,<br /> Habilidade VARCHAR(100)<br />);</code></pre> </blockquote><p>Remova os atributos <code>Código</code> e <code>Habilidade</code>.</p>
