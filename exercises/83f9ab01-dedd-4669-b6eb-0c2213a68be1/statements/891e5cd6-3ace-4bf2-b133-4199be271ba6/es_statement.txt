<h2>Eliminar columnas de una tabla</h2><p>Utilizaremos la cl&aacute;usula <code>DROP</code> del comando <code>ALTER TABLE</code>, junto con la columna que queremos eliminar. Se eliminar&aacute;n las columnas indicadas sin poder deshacer esta acci&oacute;n. Adem&aacute;s de la definici&oacute;n de la columna, se eliminar&aacute;n todos los datos que contuviera. No se puede eliminar una columna si es la &uacute;nica que forma la tabla, para ello tendremos que borrar la tabla directamente.</p><h3>Sintaxis:</h3><blockquote><pre><code>ALTER TABLE TableName DROP [ COLUMN ] [ IF EXISTS ] <em>column_name</em> [ RESTRICT | CASCADE ];</code></pre></blockquote><h3>Ejemplo:</h3><p>En el ejercicio anterior hemos generado la tabla <code>USERS</code> con la siguiente estructura:</p><blockquote><pre> table_name | column_name |     data_type     <br />------------+-------------+-------------------<br /> users      | birth_date  | date<br /> users      | course_code | integer<br /> users      | first_name  | character varying<br /> users      | last_name   | character varying<br />(4 rows)</pre></blockquote><p>Esta estructura es equivalente a la ejecuci&oacute;n de la siguiente sentencia:</p><blockquote><pre><code>CREATE TABLE USERS (<br />    FIRST_NAME VARCHAR(25),<br />    LAST_NAME VARCHAR(30),<br />    Birth_Date DATE,<br />    Course_Code INTEGER<br />);</code></pre></blockquote><p>Para eliminar los dos atributos a&ntilde;adidos en el &uacute;ltimo apartado, deber&iacute;amos ejecutar la siguiente sentencia:</p><blockquote><pre><code>ALTER TABLE USERS<br />    DROP Birth_Date,<br />    DROP Course_Code;</code></pre></blockquote><h3>Ejercicio:</h3><p>Imagina que hemos creado la siguiente tabla:</p><blockquote><pre><code>CREATE TABLE MODULES (<br />&nbsp;&nbsp;&nbsp; Name VARCHAR(100),<br />&nbsp;&nbsp;&nbsp; NumHours NUMERIC(4),<br />&nbsp;&nbsp;&nbsp; Code INTEGER,<br />&nbsp;&nbsp;&nbsp; Skill VARCHAR(100)<br />);</code></pre></blockquote><p>Elimina los atributos <code>Code</code> y <code>Skill</code>.</p>
