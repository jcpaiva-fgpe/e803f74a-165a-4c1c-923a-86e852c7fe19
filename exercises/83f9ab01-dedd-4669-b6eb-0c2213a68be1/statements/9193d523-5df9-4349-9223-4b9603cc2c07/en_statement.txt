<h2>Removing columns from a table</h2><p>We will use the <code>DROP</code> clause of the <code>ALTER TABLE</code> command, along with the column we want to remove. The indicated columns will be deleted without being able to undo this action. In addition to the definition of the column, all the data it contained will be deleted. A column cannot be deleted if it is the only one that forms the table, for this we will have to delete the table directly.</p><h3>Syntax:</h3><blockquote><pre><code>ALTER TABLE TableName DROP [ COLUMN ] [ IF EXISTS ] <em>column_name</em> [ RESTRICT | CASCADE ];</code></pre></blockquote><h3>Example:</h3><p>In the previous exercise we generated the <code>USERS</code> table with the following structure:</ p><blockquote><pre> table_name | column_name | data_type <br />------------+-------------+------------------ -<br /> users | birth_date | date<br /> users | course_code | integer<br /> users | first_name | character varying<br /> users | last_name | character varying<br />(4 rows)</pre></blockquote><p>This structure is equivalent to executing the following statement:</p><blockquote><pre><code>CREATE TABLE USERS ( <br /> FIRST_NAME VARCHAR(25),<br /> LAST_NAME VARCHAR(30),<br /> Birth_Date DATE,<br /> Course_Code INTEGER<br />);</code></pre></blockquote ><p>To remove the two attributes added in the last section, we should execute the following statement:</p><blockquote><pre><code>ALTER TABLE USERS<br /> DROP Birth_Date,<br /> DROP Course_Code ;</code></pre></blockquote><h3>Exercise:</h3><p>Imagine that we have created the following table:</p><blockquote><pre><code>CREATE TABLE MODULES ( <br /> Name VARCHAR(100),<br /> NumHours NUMERIC(4),<br /> Code INTEGER,<br /> Skill VARCHAR(100)<br />);</code></pre> </blockquote><p>Remove the <code>Code</code> and <code>Skill</code> attributes.</p>
